UserModel

--------------------------------------------------------------------------------------------------------------------------------------------------------------------

Fields:
 name	   :  Full name of the user.
 email	   :  User's email address for authentication and communication.
 password  :  Encrypted password for login security.
 phone	   :  User's phone number for verification and contact.
 addresses :  List of saved delivery addresses.
 createdAt :  Timestamp of when the user account was created.
 googleID  :  Unique ID assigned by Google OAuth (if used).
      

--------------------------------------------------------------------------------------------------------------------------------------------------------------------
--------------------------------------------------------------------------------------------------------------------------------------------------------------------

ProductMoel

--------------------------------------------------------------------------------------------------------------------------------------------------------------------

Fields: 
 name        : Name of the product (e.g., "Apple iPhone 14").
 price       : Cost of the product in currency.
 Category    : Reference to CategoryModel, specifying product category.
 stock       : Available quantity of the product in inventory.
 description : Detailed information about the product.
 image       : URL or Base64 image representation of the product.

CartModel

--------------------------------------------------------------------------------------------------------------------------------------------------------------------

Fields: 
  user      : the user who owns the cart 
  products  : A list product in the cart 
  totalPrice: total cost of everything in the cart 

--------------------------------------------------------------------------------------------------------------------------------------------------------------------
--------------------------------------------------------------------------------------------------------------------------------------------------------------------


OrederModel

--------------------------------------------------------------------------------------------------------------------------------------------------------------------

Fields:
  User       : Reference to UserModel, indicating who placed the order.
  Product    : List of ordered products.
  totalPrice : Total cost of the order, including all charges.
  address    : Delivery address chosen by the user.
  status     : Order processing status (e.g., "Pending", "Shipped", "Delivered").
  Payment    : Reference to PaymentModel, linking payment details.
  delivery   : Reference to DeliveryModel, tracking delivery info.

--------------------------------------------------------------------------------------------------------------------------------------------------------------------
--------------------------------------------------------------------------------------------------------------------------------------------------------------------

PaymentModel

--------------------------------------------------------------------------------------------------------------------------------------------------------------------

Fields: 
   Order        : Reference to OrderModel, indicating the order being paid for.
   amount       : Amount paid by the user for the order.
   method       : Payment method used (e.g., "Credit Card", "UPI").
   status       : Payment status (e.g., "Completed", "Pending", "Failed").
   transactionId: Unique identifier for tracking the payment transaction.

--------------------------------------------------------------------------------------------------------------------------------------------------------------------
--------------------------------------------------------------------------------------------------------------------------------------------------------------------

DeliveryModel

--------------------------------------------------------------------------------------------------------------------------------------------------------------------

Fields:
  Order               : Reference to OrderModel, linking the order.
deliveryBoy           : Assigned delivery person's ID or name.
status                : Delivery status (e.g., "Out for delivery", "Delivered").
trackingUrl           : URL for real-time tracking of the order.
estimatedDeliveryTime : Expected delivery time for the order.


--------------------------------------------------------------------------------------------------------------------------------------------------------------------
--------------------------------------------------------------------------------------------------------------------------------------------------------------------

AdminModel

--------------------------------------------------------------------------------------------------------------------------------------------------------------------

Fields:
  name   : Full name of the admin user.
email    : Email ID used for admin login.
password : Encrypted password for admin authentication.
role     : Admin role (e.g., "Super Admin", "Product Manager").


--------------------------------------------------------------------------------------------------------------------------------------------------------------------
--------------------------------------------------------------------------------------------------------------------------------------------------------------------

NotificationModel

--------------------------------------------------------------------------------------------------------------------------------------------------------------------

Fields:
  User    : Reference to UserModel, identifying the recipient.
message   : Notification message text.
read      : Boolean (true/false) indicating if the notification has been read.
createdAt : Timestamp when the notification was created.
